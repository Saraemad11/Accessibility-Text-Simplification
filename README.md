# Accessibility Text Simplification

## Project Overview
An AI-powered NLP system that simplifies complex text and analyzes readability using Transformer models and Streamlit.

## Features
- Text preprocessing using NLTK and SpaCy
- Readability analysis using TextStat
- TF-IDF feature extraction
- Semantic similarity measurement
- Text simplification using T5 and BART
- Interactive Streamlit interface

## Technologies Used
- Python
- NLTK
- SpaCy
- Transformers
- Scikit-learn
- Streamlit
- Pandas
- NumPy

## Dataset
CNN/DailyMail Dataset
